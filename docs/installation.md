# Installation

## From PyPI

```bash
pip install accmix
```

## From source (editable)

```bash
git clone https://github.com/yangli04/RAD-seq_EM_model.git
cd RAD-seq_EM_model
pip install -e .
```

## Build docs locally

```bash
pip install -r docs/requirements.txt
mkdocs serve
```
